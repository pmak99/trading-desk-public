# Scan Today's Earnings

Scan all earnings opportunities for today's date and show tradeable IV Crush setups.

Run the following command from the 2.0 directory:

```bash
cd /Users/prashant/PycharmProjects/Trading\ Desk/2.0 && ./trade.sh scan $(date +%Y-%m-%d)
```

After running, summarize:
1. Number of tickers with earnings today
2. Top 3 opportunities by VRP score
3. Any EXCELLENT liquidity setups
4. Recommended strategies for the best opportunities
